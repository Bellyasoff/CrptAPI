CrptApi

Реализация класса CrptApi на Java для работы с API «Честный знак».
Класс потокобезопасен и поддерживает ограничение количества запросов к API за заданный интервал времени.

Возможности

- Ограничение количества запросов (rate limit)
- Блокировка при превышении лимита
- Потокобезопасность
- Отправка документа и подписи в виде JSON

Требования

- Java 11+
- Maven (Jackson для JSON сериализации)
